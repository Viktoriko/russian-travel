# Путешествие по России
____

Проект на курсе **Яндекс.Практикум**, оптимизированный под различные десктопные и мобильные устройства.

## Используемые технологии:  
* flex-разметка
* grig'ы'
* Анимация
* Применение относительных величин
* Методолгия БЭМ в разметке документа HTML и в файловой структуре
* Работа с изображениями, ссылками, списками и блоками
